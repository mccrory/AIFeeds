h1. AIFeeds

h2. Overview

This project provides sample code and data for a tutorial series that explore the aggregation of Artificial Intelligence and Machine Learning RSS feeds by Jason Brownlee.

The tutorials are as follows:

* AIFeeds Part 1: Artificial Intelligence and Machine Learning blogs
* AIFeeds Part 2: Recent AI Articles
* AIFeeds Part 3: Intelligently filter by social impact
* AIFeeds Part 4: Automating an AIFeed aggregator
